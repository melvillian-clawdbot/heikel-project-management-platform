# Heikel Project Management Platform

A comprehensive construction and trades project management platform designed to streamline workflows for contractors, employees, clients, and vendors.

## Overview

This platform provides a complete solution for managing construction projects, including:

- Client/customer management
- Employee time tracking and project assignment
- Subcontractor and vendor coordination
- Payment processing and invoicing
- Project scheduling and work tracking
- Multi-channel communication (voice, text, video, photo)
- Document and file management

## Documentation

See [ARCHITECTURE.md](ARCHITECTURE.md) for detailed feature specifications and site architecture.

## Key Features

### For Clients
- View upcoming and past work
- Track estimates and invoices
- Upload photos and videos
- Direct communication with project managers
- Payment management

### For Employees
- Time clock in/out
- Project notes and documentation
- Work scheduling
- Direct communication with HQ

### For Subcontractors/Vendors
- RFQ (Request for Quote) management
- Work scheduling and tracking
- Invoice and payment tracking
- Project-specific permissions
